### 关于考试

考试风格不太稳定。

22春：

- 期中考试比较基础，主要是对主要模型的考察，题型比较固定。

- 期末考试偏难，有一些和中国国情有关的比较开放的大题，需要综合各个部分学到的模型，想回答得比较完善会很有挑战。

- 及格指南：重点关注每个模型和他们的推论，还有模型的原理，这样大题才能写出东西来。

- 高分指南：可以关注一些时事，想一想里面涉及到的知识点。
